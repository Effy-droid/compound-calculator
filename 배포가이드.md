# 복리 계산기 사이트 배포 가이드

Claude Code로 깃허브에 올리고 무료 도메인으로 사이트를 공개하는 단계별 안내입니다.

---

## 준비물

- **GitHub 계정** (없으면 https://github.com 에서 회원가입)
- **Claude Code 설치** (https://docs.claude.com/en/docs/claude-code/overview 참고)
- **이 폴더의 파일들** (`index.html`)

---

## 1단계 — 폴더 만들고 파일 넣기

원하는 위치에 폴더를 하나 만들어요. 예: `~/Documents/compound-calculator`

그 폴더 안에 `index.html` 파일을 복사해 넣어주세요.

---

## 2단계 — Claude Code 실행

터미널을 열고 그 폴더로 이동:

```bash
cd ~/Documents/compound-calculator
```

Claude Code 실행:

```bash
claude
```

---

## 3단계 — Claude Code에게 시키기

아래 문장을 복사해서 그대로 붙여넣으면 돼요. Claude Code가 알아서 다 해줍니다.

### 처음 한 번만 — 깃 초기화 + 첫 푸시

```
이 폴더를 깃 저장소로 만들어줘. .gitignore에 .DS_Store 같은 시스템 파일들 추가하고, README.md도 간단히 만들어줘 (제목: 복리 계산기, 설명: arcway.co.kr 스타일의 한국형 복리 계산기). 그다음 모든 파일을 커밋해줘. 커밋 메시지는 "feat: 복리 계산기 초기 버전" 으로.

그리고 GitHub CLI(gh)로 새 public 저장소를 만들어줘. 이름은 compound-calculator. 만든 다음 main 브랜치를 origin에 푸시해줘.

마지막으로 GitHub Pages를 활성화해줘. main 브랜치의 루트 디렉토리에서 서빙되도록.
```

---

```
변경된 내용 확인하고, 의미 있는 커밋 메시지로 커밋해서 GitHub에 푸시해줘.
```

## 트러블슈팅

**"gh: command not found"** 
→ GitHub CLI를 설치하세요: https://cli.github.com  
→ 설치 후 `gh auth login` 으로 로그인

**"Permission denied (publickey)"** 
→ SSH 키가 없거나 GitHub에 등록 안 됨. Claude Code에게 "SSH 키 설정 도와줘" 라고 물어보세요.

**페이지가 404 뜸**
→ Settings → Pages 에서 Source가 `main` 브랜치 `/ (root)`로 설정됐는지 확인  
→ 5~10분 더 기다려보세요
