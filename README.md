# 복리 계산기

복리는 시간과 꾸준함의 보너스입니다. arcway.co.kr 스타일의 한국형 복리 계산기입니다.

**바로 사용하기 → https://effy-droid.github.io/compound-calculator/**

## 기능

- **이대로 가면 얼마가 될까** — 원금, 적립금, 기간, 이자율로 미래 자산 예측
- **목표 금액까지 얼마나 걸릴까** — 목표를 정하면 도달까지 걸리는 시간 계산
- **72의 법칙** — 원금이 2배가 되는 시간 보너스 표시
- 다크모드 자동 지원
- 모바일 반응형

## 사용 방법

- **온라인**: 위 사이트 주소로 접속하면 됩니다.
- **로컬**: `index.html` 파일을 브라우저로 열면 됩니다.

## 기술 스택

- HTML / CSS / Vanilla JavaScript
- Chart.js (그래프)
- Pretendard 폰트

## 배포

GitHub Pages로 배포됩니다 (`main` 브랜치 루트). `main`에 푸시하면 자동으로 사이트에 반영됩니다.
